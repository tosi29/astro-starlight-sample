## 比較表

| 観点 | Starlight | Quartz |
| :--- | :--- | :--- |
| 得意領域 | 体系化されたDocs | 双方向リンク中心の庭 |
| 初見読者 | 導線を作りやすい | 迷いやすい場合がある |
| 未整理メモ | 置き場を作る必要あり | そのまま置きやすい |
| GitHub Pages | 静的出力で対応可 | 静的出力で対応可 |

## この用途での見立て

他人にも読ませたい知識ベースならStarlightが有利です。
自分用の断片メモを育てる用途ならQuartzが自然です。
