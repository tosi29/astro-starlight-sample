## 比較

| 公開方式 | 利点 | 注意点 |
| :--- | :--- | :--- |
| GitHub Pages docs/ | 設定が単純 | 生成物コミットが必要 |
| GitHub Actions Pages | 自動化しやすい | Actions設定が増える |
| Cloudflare Pages | 高機能 | 初期設定が別途必要 |

## 今回の判断

Issue要件に合わせ、`main`ブランチの`docs/`を公開する構成にしています。
