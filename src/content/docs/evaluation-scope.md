## 評価すること

このプロトタイプでは、Starlightを知識整理サイトとして使うときの手触りを確認します。

- 30ページ前後の情報を章立てできるか
- 初見読者向けの導線を作れるか
- AI会話ログを読者向け記事へ再編集しやすいか
- 技術メモ、ADR、用語集、比較表を混在させられるか
- GitHub Pagesの`docs/`公開が運用に耐えるか

## 評価しないこと

以下は、このサンプルの主目的ではありません。

- 大規模サイトでの性能測定
- 詳細なSEO検証
- 独自デザインテーマの作り込み
- 認証つき社内ナレッジベースとしての評価

## サンプルコンテンツの性格

各ページは架空の題材ですが、単なる水増しではありません。
AIログ、技術メモ、調査メモ、判断メモを実際に置くときに起きる悩みを題材にしています。
